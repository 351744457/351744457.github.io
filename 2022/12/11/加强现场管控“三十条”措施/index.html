加强现场管控“三十条”措施

2022-12-11

加强现场管控，强化直接管理，扎实执行各项制度，构建长效稳定管理机制，推进职责更细化、任务量化、流程规范、效果优化、考核准确、特别制定“三十条”措施。

一、压实责任抓点名

  1. 开锁仓：晚上对册点名、早上对册点数，锁由外向里，开由里向外
  2. 车间：暂停活动，有序组织，民J亲自组织
  3. 装卸货：加强现场巡查，“暗点”“明点”相结合
  4. 开展清点时，JQ值班领导必须到现场

二、突出重点抓巡查

  1. 2、5号岗：现场巡查，不得闲聊
  2. JQ值班领导：确保不少于50%时间到三大现场巡查，每晚🔒仓后安排2人巡查楼层走廊
  3. 指挥部巡查：各业务线负责各大现场，每批次实现全覆盖
  4. 指挥部夜巡：渔政牵头，晚上🔒仓后，各JQ宿舍现场巡查，每批次全覆盖
  5. TJ队：重点时段、重点部位的日常巡查，白天到TGTJ、医院门诊、留仓LBC现场巡查不少于一次，每晚凌晨后需巡查TGTJ至少一次

三、集中力量抓清查

  1. JQ搜身：“九个动作”落实出收工、零散（会见、看病、装卸货、提审、开庭等）搜身，严禁私自携带任何物品，须集中存放、专人清查，生产物料、生产垃圾严禁带入宿舍区
  2. JQ清查：每批次队专管包干仓和组至少一次清查，JQ领队对清查效果复核抽查
  3. 指挥部搜身：每批次由渔政牵头，指挥部全体成员参与，抽取至少两个JQ的👨‍🦲突击开展全方位、彻底搜身，逐步实现全覆盖
  4. 指挥部清查：每批次抽两个JQ车间和宿舍开展清查，车间由生产牵头，宿舍由渔政牵头，清查人力包括指挥部全体和各JQ抽取1名责任心强、清查经验丰富的领导或干事，统一由指挥部调度，逐步实现全覆盖。被抽取的派2名MJ配合。车间一般周日下午，宿舍一般周三上午

四、补齐短板抓互间

  1. 严禁脱离：任何场合任何时候严禁落单
  2. 车间现场：举手示意，至少一名组员或辅勤陪同，方可离开，严禁私自走动
  3. 宿舍活动：仓门保持半关闭状态，留一人通过的空间，设置1名临时值班在门口负责，需外出时，要想临时值班或仓长说明事由，并在至少一名组员或辅勤陪同，方可离开，不得串房
  4. JQ值班领导：实时掌握分布，零星外出，须经领导同意

五、除患务尽抓排查

  1. 异常舆情：MJ深入排查思想动态，每天1条，每批次至少2条关于思想动态的舆情
  2. 二五仔：每批次向二五仔、辅勤布置任务，落实考核
  3. 领导：JQ、FJQ全面掌握群体的整体情况、主要思想动态，有异常及时处理，交接好
  4. MJ诉求：每批次对专管开展诉求和剪举谈话，收集信息，及时解决合理诉求，不能解决向上反映，事事有回音
  5. 领导诉求大接访：每批次一次，收集集中反映较多、矛盾较尖锐的情况，及时处置解决或向上反映，每批次开展师傅与👨‍🦲接触、传递物品情况排查
  6. 指挥部：对JQ上报的诉求、研判、协调、解决或解答

六、明确责任抓教育

  1. FQ：加强思想教育引导，开展收工讲评教育，每批次对专管、全体至少2次专题教育讲评，异常时“十必谈”，强化学习现场纪律，做好“六要”“六不“
  2. JQ：督促FQ按要求开展讲评教育，每批次对全体至少一次集体讲评教育，对师傅每批次一次专题纪律教育，要求师傅与👨‍🦲接触要有陪同

七、细化任务抓考核

  1. JQ：日常管理系统上做好执勤情况登记
  2. JQ领导：做好自查自纠，发现问题，及时整改，对责任人落实考核
  3. 指挥ZX：视频考核检查
  4. 指挥嘟茶：加强监督考核，及时通报

八、强化监督抓问责

  1. 指挥部：现场管理督导考核，当场指出、做好记录、跟踪问效、整改到位，坚持”四不放过“原则，原因未查清不放过，责任人未受到处罚不放过，整改防范措施不落实不放过，相关人员未受到处罚不放过
