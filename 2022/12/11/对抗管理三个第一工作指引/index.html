对抗管理三个第一工作指引

2022-12-11

为进一步提高规范处理对抗管理事件的能力，推进规范文明管理，维护秩序稳定，有效防范风险，确保安全。

一、基本原则

（一）保持冷静：冷静面对，时刻保持心态沉稳，防止被激怒，确保作出正确处置

（二）转换环境：快速带离，防止对抗升级，事态蔓延、扩大影响

（三）形成震慑：协同处置，集中优势资源，有效压制嚣张气焰

（四）不乱动手：未威胁人身、财产、公共利益时，尽可能避免武力处置，坚持依法依规、占据主动，防止被利用

二、处置流程指引

（一）软对抗情形

  1. 录音录像：执法记录仪对准，全场录音录像，做好正剧保全
  2. 口头警告：第一次警告“立即停止，配合管理”
  • 停止对抗，命令到询问室，2人以上，当事人回避
  • 不服从，采取下一步
  1. 报告领导：回报相关情况
  2. 稳定：引导安抚，尽可能稳定情绪，了解诉求，防止矛盾激化
  3. 组织支援：召集有效人力携带家伙迅速到场支援
  4. 转换场所：命令到询问室，2人以上，当事人回避
  • 不配合：第二次警告，“配合管理，否则强制带离”
  • 不配合，指定身体素质好的2人JC强制带离，其它人现场警戒，存在对抗，依法依规使用借具压解带离
  1. 平复情绪：先平复情绪，做好安全防控
  2. 谈话教育：情绪稳定后，开展谈话教育，了解诉求矛盾，加强思想引导
  3. 后续处理：扣分处理
  4. 存在受伤：及时通报住间煎茶，受伤告知家属

（二）口头顶撞、辱骂等

  1. 录音录像：执法记录仪对准，全场录音录像，保存正剧
  2. 口头警告：现场第一次警告，“立即停止对抗，配合管理”
  • 停止的：命令到询问室，2人以上，当事人回避
  • 不停止：下去下一步
  1. 报告领导：汇报相关情况
  2. 控制现场：支援人力到达前，现场可组织表现好的👨‍🦲协助防范，防止情绪失控采取过激行为
  3. 组织支援：召集有效人力和家伙迅速到场支援
  4. 转换场所：命令到询问室，2人以上，当事人回避
  • 不配合离开：第二次警告，“配合管理，否则强制带离”
  • 不配合：指定2个身体素质相对好JC使用借具压解带离，其他人现场警戒
  1. 平复情绪：必要时采取“冷处理”，期间做好安全防护
  2. 事件调查：情绪稳定后，对本人和其他人询问，了解事情起因、经过等
  3. 谈话教育：对本人谈话教育，加强思想引导
  4. 后续处理：扣分处理
  5. 存在受伤：及时通报住间煎茶，告知家属

（三）自伤自残等方式对抗

  1. 录音录像：紧急下，先采取控制等措施，非紧急下，将执法记录仪对准，全场录音录像，做好正剧保存
  2. 伺机制止：
  • 能当场制止的，立即予以制止
  • 难以当场及时制止的，采取下一步
  1. 口头警告：第一次警告，“立即停止对抗，配合管理“
  • 停止的：命令到询问室，2人以上，当事人回避
  • 不停止的：采取下一步
  1. 报告领导；汇报情况，情况严重紧急的，报告医院做好救治准备工作携带必需用品迅速到现场
  2. 稳定：引导安抚，尽可能稳定情绪，了解诉求，防止矛盾激化
  3. 控制现场：对行为较为激烈导致严重后果，支援到达前，现场可组织表现好的👨‍🦲协助防范，防止情绪失控采取过激行为
  4. 疏散：事态较大可能影响正常活动秩序，现场组织其他人有序离开现场
  5. 组织支援：召集有效人力携带家伙迅速到现场
  6. 转换场所：命令到询问室，2人以上，当事人回避
  • 不配合：第二次警告，“配合管理，否则将依法强制带离”
  • 不配合：指定2个身体素质较好JC用借具压解带离，其它人现场警戒
  1. 检查治疗：及时检查身体，受伤较重，立即治疗，做好安全防控措施
  2. 事件调查：情绪稳定后，按要求和其他证人进行询问，了解事情起因、经过等
  3. 谈话教育：加强思想引导
  4. 后续处理：扣分处理
  5. 及时通报住间煎茶和告知家属

（四）袭击JC

  1. 伺机制服
  • 能当场制服的，立即制服依规定加带借具
  • 难以当场制服，保持适当距离，稳定情绪，保护自身和周边人员安全，采取下一步
  1. 录音录像：
  2. 口头警告：第一次；
  • 停止的
  • 不停止的
  1. 报告领导：汇报情况，事态难控制，立即报告指挥ZX，增派TJ队，其它JQ人力支援
  2. 控制现场：组织表现好的光头协助防范，无把握的，尽可能稳定情绪，等待支援
  3. 疏散：事态较大影响正常活动秩序，组织有序离开现场
  4. 组织支援
  5. 合力制服
  • 支援到位后：第二次警告
  • 不停止：使用借具制服，带离
  1. 事件调查：
  2. 谈话教育
  3. 后续处理：扣分处理，构成饭醉，按规定立案斟茶。
  4. 受伤的通报住间煎茶告知家属
